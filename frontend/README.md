# Architect Website – Frontend

## Overview

This repository contains the frontend part of the Architect Website project.
The application represents a modern portfolio website for an architect, where users can explore architectural projects, learn about the architect or company, and submit requests for services.

The frontend is responsible for providing the user interface (UI) and user experience (UX), displaying project information, navigation between pages, and handling user interactions such as form submissions.

The application is built using React and follows a component-based architecture.

---

## Features
- Displaying the main landing page
- Viewing architectural projects
- Viewing information about the architect/company
- Displaying advantages of modular houses
- FAQ section
- Contact form for service requests

## Components

The UI is divided into reusable components.

### Navbar
Navigation bar with links to the main sections of the website.

### Hero
Top section of the landing page with a large background image and main call-to-action.

### HouseVariants
Displays available house or project options.

### About
Section describing the architect or company.

### Steps
Shows the stages of cooperation and project implementation.

### BestProjects
Displays highlighted architectural projects.

### Advantages
Lists the advantages of modular houses.

### ContactForm
Allows users to submit a request for architectural services.

### FAQ
Frequently asked questions section.

### Footer
Contains navigation links and contact information.
